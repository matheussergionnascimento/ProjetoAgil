# Projeto Ágil - Sistema de Gerenciamento de Tarefas

## Objetivo
Desenvolver um sistema web para gerenciamento de tarefas utilizando metodologias ágeis.

## Escopo
O sistema permite cadastrar, listar, editar e excluir tarefas.

## Metodologia
Foi utilizada a metodologia Kanban para organização das atividades.

## Tecnologias
- Python
- Git
- GitHub
- GitHub Actions
- Pytest

## Como executar

1. Instale o Python.
2. Execute o arquivo app.py.

## Mudança de Escopo

Durante o desenvolvimento foi adicionada a funcionalidade de prioridade das tarefas.
